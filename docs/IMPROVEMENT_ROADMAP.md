# Roadmap улучшений системы сравнительного анализа

Файл фиксирует идеи развития системы и статус реализации. Его удобно использовать как технический backlog для ВКР, демонстрации и дальнейшей разработки.

## Статусы

- `[x]` реализовано в текущем прототипе
- `[~]` частично реализовано, требуется развитие
- `[ ]` запланировано
