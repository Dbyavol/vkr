# Roadmap улучшений системы сравнительного анализа

Файл фиксирует идеи развития системы и статус реализации. Его удобно использовать как технический backlog для ВКР, демонстрации и дальнейшей разработки.

## Статусы

- `[x]` реализовано в текущем прототипе
- `[~]` частично реализовано, требуется развитие
- `[ ]` запланировано

## Последние изменения
- [x] Ошибки регистрации и входа показываются прямо в модальном окне авторизации, включая дубликат email, неверный пароль и ошибки валидации полей.
- [x] Auth-service возвращает структурированные сообщения ошибок с кодом и понятным текстом для пользователя.
- [x] В истории расчетов добавлена кнопка повторного применения сценария: веса, критерии, режим анализа, целевой объект и проект восстанавливаются из сохраненных параметров.
- [x] Добавлен предпросмотр отчета перед скачиванием: HTML-preview открывается в модальном окне, а HTML-экспорт экранирует пользовательские значения.
- [x] В UI добавлены крупные интерактивные графики результатов (bar/radar/waterfall).
- [x] Реализованы пороговые серверные фильтры для analog search: числовые диапазоны и категориальные allowlist.
- [x] Добавлены сценарии устойчивости топ-N (-10%/базовый/+10%) с отображением в блоке надежности.
- [x] На этапе подготовки данных отображаются матрица пропусков и корреляционная матрица.
- [x] Добавлена поддержка datetime-полей в профилировании и предобработке с генерацией признаков year/month/day/day_of_week.
- [x] В подготовке данных добавлен выбор encoding (none/one-hot/ordinal/binary-map) прямо в UI-конфигурации полей.
- [x] Экран подготовки данных разбит на подэтапы (пропуски, выбросы, типы, encoding) с отдельной навигацией.
- [x] В блоке выбросов добавлен вывод проблемных строк по колонкам и полноразмерные графики распределений.
- [x] Добавлен PDF-экспорт отчета из UI (генерация по предпросмотру отчета) и сохранение файла отчета в S3.
- [x] В storage-service добавлен endpoint привязки report-файла к `comparison_history`, а orchestrator возвращает `history_id` в результате запуска.
- [x] В экспортируемый отчет добавлен графический блок итоговых оценок и поясняющий аналитический текст на русском языке.
